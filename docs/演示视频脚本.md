# 演示视频脚本

## 0:00-0:40 项目介绍

大家好，本项目是自然语言处理技术课程 Project，题目是“基于预训练生成模型的中文新闻标题与摘要生成系统”。系统面向中文新闻正文，可以自动生成新闻标题和新闻摘要，同时提供关键词提取、生成耗时统计、关键词覆盖率分析和用户历史记录管理。

本项目不是直接调用大模型 API，而是基于开源预训练模型 Randeng-T5 进行多任务微调，并完成了从数据处理、模型训练到前端展示的完整流程。

## 0:40-1:30 系统登录与页面说明

首先在终端中运行 `python app.py`，浏览器打开 `http://127.0.0.1:7860`。

进入系统后，可以看到顶部是项目标题和简介。页面上方提供用户名输入框，输入用户名后点击“登录”，系统会将后续生成记录保存到当前用户名下。

系统主体分为三个 Tab：“新闻生成”“我的记录”和“系统说明”。新闻生成用于模型推理，我的记录用于查看和导出历史结果，系统说明用于展示模型和实验指标。

## 1:30-3:00 新闻标题与摘要生成演示

切换到“新闻生成”页面。左侧是新闻正文输入框，系统默认提供了一段示例新闻，也可以手动粘贴新的新闻正文。

下面可以选择摘要长度，包括短、中、长三种选项；还可以通过滑块选择关键词数量。设置完成后点击“一键生成标题与摘要”。

等待模型推理完成后，右侧会显示生成标题、生成摘要、关键词和结果分析。这里可以说明标题较短，用于概括新闻主题；摘要更完整，用于保留新闻主要内容。

## 3:00-4:00 关键词与覆盖率分析

系统使用 jieba TF-IDF 从输入正文中提取关键词，并用中文顿号连接显示。

结果分析区域包括生成耗时、标题字数、摘要字数、标题关键词覆盖率和摘要关键词覆盖率。关键词覆盖率表示提取出的关键词有多少出现在生成标题或摘要中，可以辅助判断生成结果是否覆盖正文核心信息。

需要说明的是，当前关键词主要用于结果分析，并没有直接参与模型生成。后续可以扩展为关键词增强生成。

## 4:00-5:00 我的记录与导出演示

切换到“我的记录”页面，点击“刷新我的记录”，可以看到当前用户的历史生成记录。表格中展示生成时间、标题、摘要长度、关键词数量和覆盖率。

下方“最近一条详情”会展示最新记录的输入正文、生成标题、生成摘要、关键词和分析指标。

点击“导出当前用户记录为 JSON”，系统会把当前用户的历史记录导出为 JSON 文件，便于保存实验结果或提交演示材料。

## 5:00-6:00 系统说明与训练指标

切换到“系统说明”页面，介绍模型和训练信息。基础模型是 `IDEA-CCNL/Randeng-T5-77M-MultiTask-Chinese`，训练方式是标题生成和摘要生成统一建模为 Text-to-Text 多任务学习。

训练数据包括标题生成 5000 条、摘要生成 5000 条；验证数据包括标题生成 1000 条、摘要生成 1000 条；训练轮数为 1 epoch。验证指标包括 BLEU、ROUGE、ROUGE-1、ROUGE-2 和 ROUGE-L。

## 6:00-结束 总结与扩展方向

最后总结项目特点：第一，基于开源预训练模型微调，而不是调用外部大模型 API；第二，使用多任务提示词统一标题生成和摘要生成；第三，系统前端集成了关键词分析、结果分析、历史记录和导出功能。

后续可以继续扩展关键词增强生成、SQLite 数据库存储、更多训练数据、更长训练轮数和更全面的人工评价。
