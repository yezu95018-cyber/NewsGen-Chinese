# 课程提交清单

## GitHub 仓库

- GitHub 代码链接。
- 根目录 `README.md`。
- `app.py` 前端系统代码。
- `scripts/` 推理和数据处理脚本。
- `requirements.txt` 依赖文件。
- `.gitignore`，确认不会提交大数据和模型文件。

## 数据材料

- `samples/` 小样例数据。
- 完整数据说明。
- 完整训练数据本地路径或下载方式说明。

## 模型材料

- 训练好的 checkpoint 文件。
- 如果 checkpoint 过大，提供网盘链接。
- 说明模型应放入 `outputs/multitask_v2_10000/` 后再运行系统。

## 文档材料

- 项目报告：`docs/项目报告.md`。
- 使用说明文档：`docs/使用说明文档.md`。
- 演示视频脚本：`docs/演示视频脚本.md`。
- 演示 PPT 大纲：`docs/演示PPT大纲.md`。

## 演示材料

- 演示 PPT。
- 3 到 5 分钟或 5 到 8 分钟演示视频。
- 系统运行界面截图。
- 测试过程截图。
- 生成结果截图。
- 训练结果日志或 checkpoint 截图。

## 提交前检查

- 不执行 `git add .`。
- 不提交 `outputs/`。
- 不提交 `data/title_raw/`、`data/summary_raw/` 或处理后的大数据目录。
- 不提交 `.ckpt`、`.pt`、`.pth`、`.bin`、`.safetensors`。
- 不提交真实用户历史记录或导出记录。
