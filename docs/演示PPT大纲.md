# 演示 PPT 大纲

## 第 1 页：项目背景与任务要求

- 中文新闻数量增长，用户需要快速获取关键信息。
- 标题生成和摘要生成是典型自然语言生成任务。
- 课程要求提交代码、数据集、报告、说明文档、PPT 和演示视频。

## 第 2 页：系统目标

- 输入中文新闻正文。
- 自动生成新闻标题和新闻摘要。
- 提取关键词并分析生成结果。
- 支持用户历史记录查看和导出。
- 构建适合课程展示的 Gradio 系统。

## 第 3 页：技术路线

- Randeng-T5 预训练生成模型。
- Text-to-Text 统一建模。
- 多任务学习。
- jieba TF-IDF 关键词提取。
- Gradio 前端系统。
- JSON 本地历史记录存储。

## 第 4 页：模型与数据

- 基础模型：`IDEA-CCNL/Randeng-T5-77M-MultiTask-Chinese`。
- 标题训练数据：5000 条。
- 摘要训练数据：5000 条。
- 标题验证数据：1000 条。
- 摘要验证数据：1000 条。
- 模型大小：296M。

## 第 5 页：多任务生成方法

- 标题生成提示词：`生成标题：`
- 摘要生成提示词：`生成摘要：`
- 将标题和摘要样本混合训练。
- 同一个 checkpoint 根据提示词执行不同任务。

## 第 6 页：系统功能设计

- 新闻标题生成。
- 新闻摘要生成。
- 摘要长度控制。
- 关键词提取。
- 关键词覆盖率分析。
- 生成耗时统计。
- 用户登录与历史记录。
- JSON 导出。

## 第 9 页：测试结果分析

- 标题能够概括正文主题。
- 摘要能够保留新闻主要事实。
- 关键词覆盖率可辅助观察核心信息覆盖情况。
- CPU 推理耗时较长，GPU 环境可提升速度。
- 关键词目前用于分析，尚未直接参与生成。
